Entenda o projeto

O projeto tem por objetivo identifcar comentários tóxicos a partir de opinião pública, ou também conhecido por crowdsourcing.
Funciona assim: o site gera um comentário aletório para você, a partir dai é fácil, basta clicar em TÓXICO caso você considere o comentário gerado tóxico ou ofensivo ou clique em LIMPO! caso você considere que o comentário não seja tóxico ou ofensivo.
O que você decidiu irá alimentar uma base de dados que futuramente servirá para a identificação automática de comentários não desejados em páginas web!

Sobre o Projeto

Para o desenvolvimento deste website foi realizado um estudo da linguagem Python e suas bibliotecas para raspagem de dados, especificamente comentários públicos em notícias do site g1.globo.com/, a fim de realizar a classificação dos dados capturados como ofensivos ou não ofensivos e, se ofensivos, qual é a categoria (tipo de ofensa). O site é baseado no conceito de crowdsourcing, ou seja, um sistema de contribuição colaborativa ou coletiva, na qual diversos usuários, como você, visualizarão os comentários de forma aleatória e indicarão se o comentário é ofensivo ou não, assim como sua categoria, caso desejarem.
